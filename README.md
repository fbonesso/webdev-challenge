## How to Use

#### Install Docker: 
```apt-get install docker``` or ```brew install docker```

#### Clone the project:
```git clone https://github.com/fbonesso/webdev-challenge.git && cd webdev-challenge```

#### Create Docker Instance:
```docker-compose up```

#### Open the URL:
```http://localhost:8080```
